Skin Disease Classification

ABOUT THIS WEBSITE

This website leverages the power of Deep Learning by using Convolutional Neural Networks (CNN's) and hence classifies a given image into either of 8 skin diseases. I decided to learn CNN's by actual implementation, so I stumbled upon this dataset on Kaggle and thought to create a CNN and implement it using Flask. These skin diseases are:
  • Cellulitis
  • Impetigo
  • Athlete-Foot
  • Nail-Fungus
  • Ringworm
  • Cutaneous-larva-migrans
  • Chickenpox
  • Shingles
